This agent is part of the Concierge Agents track. It acts as an automated learning partner: it gathers your goals, designs a structured learning path, delivers a high-impact daily summary, and evaluates your understanding through immediate, gamified quizzes.

[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's look at the system architecture. The system starts at the Root AI Agent. The Root Agent orchestrates the flow. When a user creates or updates their configuration, the agent invokes the save_user_profile tool to store their goals and schedule. Daily lesson delivery is driven by the search_simulated_kb tool, which queries a structured knowledge index to fetch target content. After reading, the user completes a quiz, and the log_quiz_result tool records the performance metrics. All user preferences and scores are loaded back via the get_user_profile tool.

[3:00 — SECURITY & MCP]
