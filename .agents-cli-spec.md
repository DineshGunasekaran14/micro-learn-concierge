# Agent Spec: Micro-Learning Concierge

## Overview
The **Micro-Learning Concierge** is a personalized educational assistant that curates custom, 5-minute daily lessons tailored to a user's specific learning goals, schedules these lessons, and tests comprehension via short interactive quizzes.

## Example Use Cases
1. **Goal Setting & Scheduling:**
   - **Input:** User sets a goal: "Learn the basics of Quantum Computing" to be scheduled daily at "08:00 AM".
   - **Output:** Agent saves the goal and schedule, confirming: "Your learning plan for 'basics of Quantum Computing' is set for daily delivery at 08:00 AM."
2. **Daily Lesson Delivery:**
   - **Input:** User requests: "Give me today's lesson."
   - **Output:** Agent retrieves/generates a custom 5-minute lesson (around 500 words) on a sub-topic (e.g., "Introduction to Qubits"), followed by a 3-question multiple-choice quiz.
3. **Interactive Quiz & Review:**
   - **Input:** User answers the quiz questions.
   - **Output:** Agent evaluates the answers, provides immediate feedback/explanations, and notes the user's progress.

## Tools Required
1. `save_user_profile(goals: list[str], daily_schedule_time: str) -> str`: Saves the user's learning goals and preferred daily delivery time to the session state.
2. `get_user_profile() -> dict`: Retrieves the current user's profile details.
3. `search_simulated_kb(topic: str) -> str`: Simulates searching an offline database/knowledge base for reference materials to support offline sandbox execution.
4. `log_quiz_result(score: int, total: int, topic: str) -> str`: Records the user's performance on the topic quiz.

## Constraints & Safety Rules
- **Length Constraint:** Curated lessons must be readable in 5 minutes (strictly between 400 and 600 words).
- **Educational Scope:** Avoid generating non-educational content; focus exclusively on the designated learning goals.
- **Safety:** Filter out harmful, unsafe, or inappropriate topics.

## Success Criteria
- Daily lessons generated correspond exactly to the saved learning goals.
- Lessons include interactive quizzes that test the key concepts.
- User profile and quiz history are correctly persisted across the session state.
